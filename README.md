# NAND2Tetris in Virdant

[NAND2Tetris](https://www.nand2tetris.org/) is an educational project where you build
a 16-bit CPU and computer, starting with the most basic electronic component, the NAND gate,
and finishing with a computer that can play the classic game of Tetris.

[Virdant ](https://virdant.org/) is a modern hardware description language (HDL) that
is meant to make hardware design pleasant and to serve as a readable backend target
for hardware generators.
